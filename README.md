# 小英雄成長大冒險 🚀

> 兒童行為管理 & 獎勵系統 Web App

## 🌐 線上使用
https://ken0900101318-beep.github.io/hero-adventure/

## ✨ 功能特色

### 📋 任務系統
- **6 個日常任務**：刷牙、家事、考試100分、整潔、準時睡覺、跳繩
- 點擊任務按鈕即可獲得點數
- 每個任務都有精美的視覺反饋和音效

### ⭐ 點數與進度
- **21 點進度系統**：發光的進度球動態顯示
- 點數增加時有發光動畫效果
- 達到里程碑（7, 14, 21分）會有特殊獎勵

### 🏆 獎勵寶箱
- **7 分**、**14 分**、**21 分**分別解鎖一個寶箱
- 解鎖後可以手寫輸入獎勵內容
- 寶箱解鎖時有華麗的視覺效果

### 🛡️ 守護盾系統
- **5 個守護盾**
- 一般違規行為扣除一個
- 盾牌全失時會有警告提示

### 🌀 黑洞封印模式（家長管理）

**⚠️ 重要功能：雙重保護機制**

#### 啟動封印
1. 點擊「啟動封印」按鈕
2. **首次使用**：系統會要求設定 4 位數密碼（請妥善保管！）
3. 輸入鎖定時長（小時），例如：24
4. 確認後封印啟動：
   - 所有任務按鈕變灰鎖死
   - 無法獲得點數
   - 整個畫面變暗

#### 解除封印
有兩種方式：

**方式 1：自動解除**
- 等待設定的時間到期
- 系統會自動解除封印

**方式 2：家長提前解除**
- 點擊按鈕（會顯示剩餘時間）
- 輸入正確的家長密碼
- 確認後即可提前解除

**安全性：**
- 小孩不知道密碼就無法自行解除
- 適合作為「嚴重違規」的懲罰措施
- 密碼會保存在本機，除非清除瀏覽器資料

### ↶ 撤銷功能
- 右下角綠色圓形按鈕
- 點擊可撤銷上一步操作（誤觸取消）
- 保留最近 10 次操作歷史
- 沒有歷史時按鈕會變灰

### 💾 自動保存
- 所有數據自動保存在瀏覽器本機（LocalStorage）
- 關閉網頁後重新打開，進度依然保留
- 即使離線也能正常使用

### 🔊 音效反饋
- 使用 Web Audio API 生成內建音效
- 完全離線，無外部連結
- 加分、解鎖、扣盾、黑洞等都有不同音效

## 💡 使用建議

### 給家長的建議：

1. **設定清晰的獎勵**：在寶箱裡填入具體的獎勵（例如：去遊樂園玩、買玩具等）
2. **一致性很重要**：每次完成任務都要點擊，建立習慣
3. **黑洞封印謹慎使用**：只用於嚴重違規，保持其「特殊性」
4. **密碼保管好**：設定後請記住，忘記密碼只能等時間到期
5. **定期重置**：可以每週或每月重置一次進度，重新開始

### 給小朋友的建議：

1. 每天努力完成任務
2. 集滿點數就能解鎖獎勵
3. 保護好你的守護盾
4. 如果誤觸可以用撤銷按鈕
